Kỹ năng

Tóm tắt kỹ năng kỹ thuật, công cụ, chứng chỉ và mục tiêu học tập hiện tại.

Kỹ năng kỹ thuật

  • HTML

    Biết xây dựng cấu trúc trang web semantic, sử dụng đúng heading, section, article, form element và các thẻ formatting cơ bản.

    Trình độ: Nâng cao

  • CSS

    Có thể styling layout bằng Flexbox, Grid, responsive design và tổ chức class theo BEM.

    Trình độ: Nâng cao

  • JavaScript

    Biết xử lý DOM, event, form validation và xây dựng các tương tác cơ bản cho website.

    Trình độ: Trung cấp

  • Responsive Design

    Thiết kế giao diện thích nghi với nhiều kích thước màn hình như mobile, tablet và desktop.

    Trình độ: Trung cấp

  • Accessibility

    Hiểu các khái niệm cơ bản như alt text, aria-label, semantic HTML và keyboard-friendly UI.

    Trình độ: Sơ cấp

  • Git

    Biết sử dụng Git để quản lý source code, commit thay đổi và làm việc với repository cơ bản.

    Trình độ: Trung cấp
