Dự án Frontend

  • Portfolio cá nhân

    Đây là project portfolio cá nhân giúp tôi luyện tập cách xây dựng website nhiều trang với cấu trúc semantic HTML rõ ràng.

    Trong project này, tôi luyện tập cách chia page thành các section hợp lý, sử dụng BEM để đặt tên class, đồng thời đảm bảo navigation và footer thống nhất giữa các trang.

    Công nghệ sử dụng

    • HTML
    • CSS
    • JavaScript

    Tính năng chính

    • Trang chủ giới thiệu tổng quan.
    • Các trang About, Skills, Projects, Blog và Liên hệ.
    • Class naming theo BEM để dễ maintain.
    • Responsive layout cho nhiều kích thước màn hình.

  • Landing page sản phẩm

    Landing page sản phẩm là project luyện tập cách xây dựng giao diện giới thiệu sản phẩm với bố cục rõ ràng, phần hero nổi bật, danh sách tính năng và khu vực kêu gọi hành động.

    Project giúp tôi hiểu rõ hơn về spacing, visual hierarchy, button style và cách tổ chức nội dung để người dùng dễ đọc, dễ hiểu.

    Công nghệ sử dụng

    • HTML
    • CSS
    • Flexbox

    Tính năng chính

    • Hero section với tiêu đề và CTA.
    • Danh sách tính năng sản phẩm.
    • Layout responsive cho mobile.
    • Footer chứa social links.

  • Ứng dụng Todo List

    Ứng dụng Todo List là project giúp tôi luyện tập JavaScript cơ bản thông qua việc thêm, xoá và đánh dấu công việc. Đây là project nhỏ nhưng rất hữu ích để hiểu DOM và event.

    Tôi cũng học cách chia nhỏ UI thành các phần như form nhập liệu, danh sách task và trạng thái hoàn thành để code dễ đọc hơn.

    Công nghệ sử dụng

    • HTML
    • CSS
    • JavaScript
    • DOM

    Tính năng chính

    • Thêm task mới.
    • Xoá task khỏi danh sách.
    • Đánh dấu task đã hoàn thành.
    • Lưu trạng thái task bằng localStorage.

Dự án Mobile

  • Mobile Ứng dụng thời tiết UI

    Mobile Ứng dụng thời tiết UI là project luyện tập thiết kế giao diện mobile cho ứng dụng thời tiết. Project tập trung vào card layout, icon thời tiết và cách hiển thị thông tin ngắn gọn.

    Giao diện được xây dựng theo hướng đơn giản, dễ nhìn và phù hợp với màn hình nhỏ. Đây là project giúp tôi luyện thêm responsive design và UI composition.

    Công nghệ sử dụng

    • HTML
    • CSS
    • Responsive Design

    Tính năng chính

    • Giao diện tối ưu cho mobile.
    • Card hiển thị nhiệt độ và trạng thái thời tiết.
    • Thiết kế đơn giản, dễ scan thông tin.
    • Danh sách dự báo theo ngày.
